# 🏫 EM Deocleciano De Oliveira

![D.O logo](./readme_src/do_logo.png)

Bem-vindo ao repositório oficial do site da **Escola Municipal Deocleciano De Oliveira**!  
Este projeto foi desenvolvido com **React + TypeScript + Vite**, utilizando **Sass** para estilização.

---

## ✨ Sobre o Projeto

O objetivo deste site é oferecer uma plataforma moderna e acessível para alunos, professores e comunidade escolar.  
Aqui você encontrará informações sobre a escola e muito mais.

---

## 🛠️ Tecnologias Utilizadas

![Vite](./readme_src/vite.svg)
![React](./readme_src/react.svg)

- ⚛️ **React (TSX)** — construção de interfaces dinâmicas
- ⚡ **Vite** — bundler rápido e eficiente
- 🎨 **Sass (SCSS)** — estilização modular e organizada
- 📂 **TypeScript** — tipagem estática para maior segurança no código
